# Fyremezzonine Conference WordPress Theme

Одностраничная WordPress-тема под сайт конференции «Пожарная безопасность складских помещений с мезонинным хранением».

## Установка

1. Скопируйте папку `fyremezzonine-wp-theme` в `wp-content/themes/`.
2. В админке WordPress откройте `Внешний вид -> Темы`.
3. Активируйте тему `Fyremezzonine Conference`.
4. В `Внешний вид -> Настроить -> Conference Links` задайте ссылки:
   - регистрация;
   - программа конференции;
   - требования к оформлению материалов;
   - заявка партнера;
   - сайт филиала.
5. В `Внешний вид -> Меню` можно создать меню с якорями:
   - `#about`
   - `#participation`
   - `#registration`
   - `#partners`
   - `#contacts`

## Примечание

Тема не зависит от Tilda. Контент, структура и визуальная логика собраны как самостоятельный WordPress-шаблон.
